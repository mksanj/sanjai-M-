Earthquake Stabilized Dialysis System for Patient Safety During Seismic Events
Earthquake Stabilized Dialysis System is an advanced safety platform designed to protect dialysis patients and medical equipment during earthquake situations. The system automatically detects seismic vibrations and provides stabilization support to dialysis machines, patient beds, and critical medical components to reduce risks during emergencies.


The platform uses sensors, automated control mechanisms, and real-time monitoring to ensure continuous patient safety. It helps healthcare professionals receive alerts, monitor equipment conditions, and take quick actions during seismic events.

Installation

Clone Repository

git clone https://github.com/yourusername/EarthquakeDialysisSafetySystem.git

Navigate Project Folder

cd EarthquakeDialysisSafetySystem

Create Virtual Environment
python -m venv ven

Install Dependencies

pip install -r requirements.txt

Configure Database

Create MySQL database
Import database.sql file
Update database configuration in app.py

Run Application

python app.py
